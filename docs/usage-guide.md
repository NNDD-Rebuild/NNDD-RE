# 使い方ガイド

## 入手方法

[GitHub Releases](https://github.com/NNDD-Rebuild/NNDD-RE/releases) から最新版をダウンロードしてください。

- Windows: `NNDD-RE-<version>-setup.exe`（NSISインストーラ）

## 1. 初回ログイン

1. 設定 → 全般 → 「ブラウザでログイン」をクリック
2. 表示されるウィンドウで通常通りニコニコ動画にログイン
3. Cookieが自動的にNNDDに取り込まれる

## 2. 動画をダウンロードする

- DLリストタブで動画ID（sm12345等）を入力 → 「DLリストに追加」
- または検索/ランキング/マイリストから「DL」ボタン
- yt-dlp が動画・音声を取得し、FFmpegで MP4に結合してライブラリへ登録
- 初回は設定 → 外部ツール画面での yt-dlp/FFmpeg のダウンロードが必要です（詳細は [トラブルシューティング](/NNDD-RE/troubleshooting/) 参照）

## 3. 動画を再生する

- ライブラリで「再生」をクリック（ローカル再生）
- 検索/ランキング/マイリストで「再生」（ストリーミング、yt-dlp/FFmpeg不要）
- コメントは自動的に重畳表示されます

## 4. スケジュールで自動ダウンロードする

1. マイリストを登録する
2. 設定 → スケジュール → 新規スケジュール
3. 曜日と時刻を指定すると、その時刻に自動でマイリストを更新し新着動画をダウンロードします
